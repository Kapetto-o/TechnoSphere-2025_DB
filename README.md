#
**ТехноСфера** — это интернет-магазина бытовой техники созданный в рамках курсовых проектов.
**TechnoSphere-2025_DB** — это база данных интернет-магазина бытовой техники, созданная в рамках курсового проекта 5 семестра по дисциплине "Базы Данных".
___
### Задачи проекта:
* ✅ База данных должна быть реализована в СУБД MS SQL Server
* ✅ Доступ к данным должен осуществляться только через соответствующие процедуры
* ✅ Количество объектов БД (таблиц, представлений, индексов, пользователей и пр.) регламентируется задачей
* ✅ Должен быть проведен импорт данных из JSON файлов, экспорт данных в формат JSON
* ✅ Необходимо протестировать производительность базы данных на таблице, содержащей не менее 100 000 строк, и внести изменения в структуру в случае необходимости
* ✅ Необходимо проанализировать планы запросов к таблице
* ✅ Применить технологию базы данных "Резервное копирование и восстановление"
